Allow your users to continue accessing your content even while offline.

== Description ==
WordPress Offline plugin uses new [ServiceWorker](https://developer.mozilla.org/en-US/docs/Web/API/Service_Worker_API) and [Cache API](https://developer.mozilla.org/en-US/docs/Web/API/Cache) to allow your users to read the contents of your site while they are offline or under unreliable network situations by caching part of your site.

== Installation ==
1. Clone or download the project repository
2. Copy `wp-offline` directory inside your WordPress installation plugins directory
3. Enable the plugin from the admin panel
